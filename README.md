# Artemis Financial Security Project

Security analysis and secure coding practices for Artemis Financial, including vulnerability assessment and mitigation strategies.

## Overview

This project focuses on identifying and addressing software security vulnerabilities for Artemis Financial. The goal was to analyze risks within the system and apply secure coding practices to improve the protection of sensitive financial data.

## Reflection

Artemis Financial is a company that needed help identifying and fixing security vulnerabilities in their software. Their main concern was ensuring that sensitive financial data remained secure while keeping the application functional and reliable.

During the vulnerability assessment, I did well at identifying potential risks in the system and understanding how weaknesses in the code could lead to larger security issues. Coding securely is important because it protects user data, prevents cyberattacks, and helps maintain trust between a company and its clients. Strong security adds value by reducing risk and protecting the company’s reputation.

One of the more challenging parts of the assessment was determining which vulnerabilities were the highest priority. However, this was also helpful because it improved my ability to think critically and approach problems from both a developer and attacker perspective.

To increase layers of security, I applied secure coding techniques such as improving encryption and minimizing potential vulnerabilities in the system. In the future, I would use tools like static code analysis and vulnerability scanning to better assess risks and choose appropriate mitigation strategies.

To ensure the application remained functional and secure, I tested the code after making changes and reviewed it carefully to avoid introducing new vulnerabilities. This process is important because changes meant to fix issues can sometimes create new ones if not properly tested.

The tools and practices I used in this project, such as secure coding principles and testing methods, will be helpful in future assignments and real-world applications.
